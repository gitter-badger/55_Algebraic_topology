H-space


A topological space X is said to be an H-space (or Hopf-space) if there exists a continuous binary operation φ:X×X→X and a point p∈X such that the functions X→X defined by x↦φ⁢(p,x) and x↦φ⁢(x,p) are both homotopic to the identity map via homotopies that leave p fixed. The element p is sometimes referred to as an ‘identity’, although it need not be an identity element in the usual sense. Note that the definition implies that φ⁢(p,p)=p.

Topological groups are examples of H-spaces.

If X is an H-space with ‘identity’ p, then the fundamental group π1⁢(X,p) is abelian. (However, it is possible for the fundamental group to be non-abelian for other choices of basepoint, if X is not path-connected.)
